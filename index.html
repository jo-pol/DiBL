DiBL

A patchwork of sub-projects related to: Diagrams for Bobbin Lace.

  • Plain as well as decorated polar grids.
    • The web version has more patterns that scale better, example in red.
    • The InkScape version allows matching nested grids, example in blue.
  • Choose stitches for computer generated grounds and show the flow of individual threads. Both variants are web applications for modern desk-top browsers.
    • The tensioned version shows matching color-coded diagrams and thread diagrams next to one another. Whatever will improve soon, Internet Explorer is not very likely to show color codes in the foreseeable future.
    • The tiled version supports a limited set of stitches, lacks footsides and thread diagrams are only available for a subset of patterns. Painting threads requires a printer, pencil and tracing paper or an InkScape plug-in to apply thread styles to a downloaded diagram.
