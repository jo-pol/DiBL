DiBL

A patchwork of sub-projects related to:

Diagrams for Bobbin Lace.

  • Plain as well as decorated polar grids
  • Choose stitches for computer generated grounds (has some known bugs)
  • Analyse the flow of threads in a generated ground: trace the threads on paper. Once downloading is properly implemented the thread style plug-in for InkScape can do the tracing.
  • More InkScape plug-ins for grids and grounds, a screen shot shows all dialogs together.
