DiBL

A patchwork of sub-projects related to: Diagrams for Bobbin Lace.

  • Plain as well as decorated polar grids.
    • The web version has more patterns that scale better, example in red.
    • The InkScape version allows matching nested grids, example in blue.
  • Choose stitches for computer generated grounds. Once you configured an appealing thread diagram, the matching color coded pair diagram is only two clicks away. May be very slow on mobile devices.
  • Analyse the flow of threads in a generated thread diagram: trace the threads on paper or use the thread style plug-in for InkScape to do the tracing.
  • A screen shot shows all dialogs of InkScape plug-ins dedicated to bobbin lace making.
