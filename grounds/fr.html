ct/tc?

Vous aurez certainement remarqué que les descriptions des points "ouverts" ou "demi passées" sont "TC" (Tordre-Croiser)
alors que certaines dentellières utilisent plutôt "CT" (Croiser-Tordre). Ce n'est pas une erreur !
Il semblerait, d'après les recherches effectuées par Véronika Irvine, que cela dépende de la région d'origine de la dentelle
et du type de carreau utilisé : le plus souvent, un carreau en forme de traversin => "TC", un carreau plat => "CT".
Un choix devait être fait pour développer ces applications mais cela ne modifie pas profondément les résultats.

Exemple

Un bouton de soumission n'est pas encore implémenté. Pour le moment, vous devez installer DiBL et copier/coller la commande configurée dans un fichier .sh ou .bat pour l'exécuter.

  • Trouver les dernières occurrences de "tiles"
  • Téléchargez et décompacter le zip (Windows) ou tar.gz (environ 10% plus petit sous Mac/Linux seulement).
  • Déposez les fichiers .sh ou .bat dans le dossier décompressé. Par exemple: mes documents\dibl-cmd-0.1.2-bin\dibl-cmd-0.1.2

Voyez les conseils d'édition pour des images svg, par exemple, pour varier la largeur ou la couleur du fil.

Command files

  • sh (Mac / Unix, la phrase multiligne nécessite des apostrophes)
    java -jar dibl-cmd-${project.version}.jar -ext svg '3;2

tc;tc
tc;tc' < input/flanders.svg > diagram.svg

    java -jar dibl-cmd-${project.version}.jar -ext svg 3;2^
