A submit button to generate the configured diagram is not yet implemented. For now you have to install DiBL and copy/paste the configured commands in .sh or .bat files to execute.

  • Find the latest tiles release
  • Download and unpack the zip (Windows) or tar.gz (roughly 10% smaller but Mac/Unix only).
  • Put the .sh or .bat in the unzipped folder. For example: my documents\dibl-cmd-0.1.2-bin\dibl-cmd-0.1.2

See the user guide to edit SVG results, for example to vary thread width and color.

Command files

  • sh (Mac / Unix, note that the multiline string requires quotes)
    java -jar dibl-cmd-${project.version}.jar -ext svg '3;2
tc;tc
tc;tc
tc;tc' < input/flanders.svg > diagram.svg
  • bat (Windows, note that the multiline string requires empty lines)
    java -jar dibl-cmd-${project.version}.jar -ext svg 3;2^

tc;tc^

tc;tc^

tc;tc < input/flanders.svg > diagram.svg
